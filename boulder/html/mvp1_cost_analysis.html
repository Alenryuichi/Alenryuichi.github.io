ClimbEdge MVP1 - 成本分析与服务选型

ClimbEdge MVP1

成本分析与服务选型方案

目标：2024年6月30日上线 · 最小化成本 · 最大化免费资源

🚨 紧急上线计划

上线时间：2024年6月30日（距离现在约30天）

核心策略：优先使用免费服务，快速验证产品可行性

成本目标：月成本控制在¥500以内

📋 MVP1 功能范围

🎥 视频功能

  • 视频上传与播放
  • 视频分类管理
  • 基础搜索功能
  • 视频评论系统

👤 用户系统

  • 用户注册登录
  • 个人资料管理
  • 观看历史记录
  • 收藏功能

📚 术语解释

  • 攀岩术语词典
  • 视频内术语标注
  • 术语搜索功能
  • 术语分类展示

💰 MVP1 成本分析

服务选型策略

💡 免费优先原则

  • 云服务：优先使用各大云厂商的免费额度
  • 开源方案：能自建的服务尽量自建
  • 第三方服务：选择有免费套餐的服务
  • 付费服务：只购买核心必需的服务

免费服务配置详情

🖥️ 腾讯云轻量服务器

  • 配置：2核2G内存，40GB SSD
  • 时长：已有资源，2025年7月到期
  • 系统：宝塔Linux面板
  • 流量：200GB/月（3Mbps带宽）
  • 用途：部署后端服务+Redis缓存

🗄️ 腾讯云MySQL数据库

  • 配置：1核2G内存，50GB存储
  • 架构：双节点高可用
  • 版本：MySQL 8.0
  • 网络：VPC私有网络
  • 用途：用户数据+业务数据存储

📦 腾讯云COS存储服务

  • 计费模式：按量计费（推荐）
  • 存储费用：¥0.118/GB/月
  • 外网流量：¥0.5/GB
  • CDN回源：¥0.15/GB
  • 请求费用：¥0.01/万次
  • 免费额度：50GB存储+10GB流量（6个月）
  • 预估成本：¥7/月（COS）+¥10/月（CDN）

🌐 腾讯云CDN加速服务

  • 套餐：¥9.9/月基础套餐
  • 流量：50GB/月加速流量
  • 请求：300万次/月加速请求
  • 节点：国内加速节点
  • 用途：视频播放加速

👤 用户注册登录服务

  • 微信登录：微信开放平台SDK
  • 苹果登录：Apple Sign In集成
  • 手机验证：腾讯云短信服务
  • 推送服务：极光推送免费版
  • 用途：用户身份认证+消息推送

💰 COS存储成本详细分析

业务需求评估

📹 视频存储需求

  • 视频规格：1080P，5-10分钟/个
  • 文件大小：70-80MB/个（压缩后）
  • 官方视频：50-80个教学视频
  • 岩馆合作：20-30个/月
  • 存储预估：8-12GB（前3个月）

🌐 流量使用预估

  • 目标用户：100-500 活跃用户
  • 观看频次：8-15个视频/用户/月
  • 单次流量：75MB
  • 外网流量：3-28GB
  • CDN回源：6-56GB
  • CDN加速：50GB/月套餐流量

🔄 请求量分析

  • 读请求：视频播放、缩略图加载
  • 写请求：管理端上传、元数据更新
  • 月读请求：20-40万次
  • 月写请求：1-2万次
  • 费用极低：¥0.01/万次

计费方案对比

计费方式 存储费用 流量费用 CDN费用 月总费用 适用场景
COS按量+CDN套餐（推荐） ¥1.18（10GB） ¥4（外网5GB+回源10GB） ¥9.9（50GB套餐） ¥15.08 适合初期，CDN有保障
存储包+CDN套餐 ¥7.75（100GB/年） ¥4（按量） ¥9.9（50GB套餐） ¥21.65 存储量大且稳定
全按量计费 ¥1.18（按量） ¥4（按量） ¥0（不使用CDN） ¥5.18 成本最低但体验差
CDN按量计费 ¥1.18（按量） ¥4（按量） ¥15+（按量CDN） ¥20+ 流量不可预测

渐进式成本策略

🚀 阶段一：MVP1启动

时间：前3个月
策略：全部按量计费
成本：¥15-20/月
优势：灵活，风险低

📈 阶段二：用户增长

时间：3-6个月
策略：流量>30GB考虑流量包
成本：¥20-35/月
优势：成本可控

🎯 阶段三：稳定运营

时间：6个月后
策略：存储>50GB考虑存储包
成本：¥25-50/月
优势：成本最优

成本优化策略

🛠️ 技术优化措施

存储优化
  • H.264编码，控制码率
  • 多分辨率存储（720P/1080P）
  • 缩略图单独存储
  • 定期清理无效文件
流量优化
  • 启用CDN免费额度（10GB/月）
  • 配置缓存策略，减少回源
  • 使用WebP格式图片
  • 启用Gzip压缩

⚠️ 风险控制

免费资源风险

🛡️ 风险应对策略

  • 轻量服务器到期：2025年7月到期，有充足时间规划升级
  • 存储超限：压缩视频质量，使用多个COS账号
  • 流量超限：限制视频分辨率，优化缓存策略
  • 性能不足：代码优化，数据库索引优化

备用方案

风险场景 备用方案 额外成本 实施难度
轻量服务器到期 续费或升级到CVM ¥80/月 低
存储费用超预期 购买100GB存储包 ¥7.75/月 低
流量费用超预期 购买100GB流量包 ¥33/月 低
CDN流量超50GB 升级CDN套餐或按量 ¥20-50/月 低
数据库存储不足 扩容存储空间 ¥50/月 低

🎯 总结与建议

💰 成本优势

  • 初期成本：仅¥204/月，包含CDN加速服务
  • 现有资源：轻量服务器+MySQL数据库已就绪
  • 按量计费：COS存储按实际使用付费，避免浪费
  • 免费服务：微信登录+苹果登录+1000条短信/月
  • 渐进式成本：随用户增长逐步优化计费方案
  • 风险可控：即使全部付费也不超过¥400/月

🚀 立即行动计划
